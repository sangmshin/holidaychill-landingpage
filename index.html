how to set up

your echo dot

If this is your first time setting up your Echo Dot, unwrap your Dot and find the small instructions card in the Amazon box. Follow those instructions to get your Echo Dot ready to go.

If you already have an Alexa-compatible device, all you have to do to get started is say: “Alexa, enable Holiday Chill.”

how to enable

holiday chill

using a web browser:

  1. Visit the Alexa Skills Homepage
  2. Search for “Holiday Chill.” Click on Holiday Chill icon
  3. Select the “Enable” button
  4. Say this phrase for Alexa to activate the skill: “Alexa, start Holiday Chill”

Using the Alexa mobile App:

  1. Download/open the Alexa app on your mobile device
  2. Navigate to the Skills tab, located in the menu on the upper left corner of the screen
  3. Type “Holiday Chill” in the search bar
  4. Select Enable
  5. Say this phrase for Alexa to activate the skill: “Alexa, start Holiday Chill”

Using Voice:

  1. Say “Alexa, enable Holiday Chill”
  2. Your device should confirm that the skill was added

how to use the

holiday chill

using a web browser:

Chillin’ is easy. Anytime you want to use this skill, say “Alexa, start Holiday Chill.” Just follow the prompts from Alexa, responding verbally to her questions, and you’ll be on your way to a happier, healthier holiday.
